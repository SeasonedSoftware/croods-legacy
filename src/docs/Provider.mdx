# Provider

The Provider component is used to connect other components to the state managed by croods.

## API

| Property      | Type          | Required | Default    | Description                                                                                  |
| ------------- | ------------- | -------- | ---------- | -------------------------------------------------------------------------------------------- |
| baseUrl       | String        | true     | -          | It defines the api url which all other paths used in croods components will be relative too. |
| credentials   | String        | true     | -          | Credentials                                                                                  |
| headers       | String/Number | true     | -          | the entry to be destroyed, passing it in the url defined in path                             |
| renderLoading | Func          | false    | -          | A default function to be called when loading a component without a custom renderLoading      |
| renderError   | Func          | false    | -          | A default function to be called when loading a component without a custom renderError        |
| afterSuccess  | Func          | false    | -          | Parent element's Id, for more specificity in changes.                                        |
| afterFailure  | Func          | false    | -          | Parent element's Id, for more specificity in changes.                                        |
| afterResponse | Func          | false    | -          | Parent element's Id, for more specificity in changes.                                        |
| disableCache  | Bool          | false    | false      | Ex: /foo/bar-\_12?&=                                                                         |
| debugRequests | Boll          | false    | `/${name}` | If set to true, will show                                                                    |
| parseResponse | String        | false    | `/${name}` | Ex: /foo/bar-\_12?&=                                                                         |

## How to use

```javascript
// src/App.js

import { Provider as CroodsProvider } from 'croods'
import { Provider as ReduxProvider } from 'react-redux'

import store from './store'

export default props => (
  <ReduxProvider store={store}>
    <CroodsProvider baseUrl="https://some.org/api">
      // Whole app goes here
    </CroodsProvider>
  </ReduxProvider>
)
```
